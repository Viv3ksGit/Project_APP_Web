Sloka Sabha Fallback Shell

This is not the full web app. It is a small fallback page used only when the Android wrapper has no deployed app URL configured.

For the real app with CSS, fonts, images, audio, and video, run npm ci and npm run dev, then open http://127.0.0.1:3000.

For Android, set CAPACITOR_APP_URL to the deployed web app URL before running npm run android:sync.
